# 森のカフェ — ランディングページ（デモ）

## 概要

カフェ「**森のカフェ**」向けの**単一 HTML**ランディングページです。フレームワークは使わず、`index.html` 内に CSS をまとめた静的サイトです。店名・住所・営業情報は**架空のデモ用**です。

ワークスペース直下から本 README を指し示す入口は [CAFE.md](../CAFE.md) です。店名・営業時間・メニューなどの一覧は [基本情報.md](基本情報.md) を参照してください。

## ページメタ（`index.html` 実装と同期）

| 要素 | 現在の内容 |
|------|------------|
| `<title>` | 森のカフェ — 静けさと香りのひととき |
| `meta name="description"` | 森のカフェ。路地裏の小さな焙煎所と喫茶。朝の一杯から夜のデザートまで、豆と時間に誠実に向き合うカフェ。 |

文言を変えたら、この表もあわせて更新してください。

## コンセプトと訴求

| 項目 | 内容 |
|------|------|
| 店名 | 森のカフェ（東京・谷中イメージの文言） |
| 主なターゲット | **ファミリー層**（お子さま連れ歓迎、キッズドリンク・シェア向きスイーツ・デカフェなど） |
| トーン | ヒーロー周りはカジュアルな文体。ストーリー（#story）は焙煎・ドリップへのこだわりを説明するやや専門的なトーン。 |

## ページ構成（セクション）

1. **固定ナビ** — ロゴ、ことば / お品書き / 店舗へのアンカー
2. **ヒーロー** — ファミリー向けキャッチ、リード、来店・メニューへの CTA
3. **ことば（#story）** — 焙煎と抽出（ハンドドリップ・ネル・プレス）の紹介
4. **お品書き（#menu）** — ダーク背景の 3 カラム（ママ・パパの一杯 / キッズ＆ジュニア / みんなでシェア）
5. **店舗（#visit）** — 営業時間・所在地（デモ用プレースホルダ）
6. **フッター** — ブランド名・デモ表記

## デザイン

- **配色**: クリーム／紙色ベース、テラコッタ系アクセント、深緑のヒーロー視覚エリア
- **フォント**: Google Fonts — Cormorant Garamond（ディスプレイ）、Zen Kaku Gothic New（本文）
- **演出**: 軽いフィルムノイズ風オーバーレイ、ヒーローのフェードイン、ホバー時のカード浮き

## 技術

- HTML5、埋め込み CSS（外部 JS なし）
- フォントは CDN（fonts.googleapis.com）読み込み

## ローカルでのプレビュー

リポジトリ（ワークスペース）直下から:

```bash
npm run preview:cafe
```

ブラウザで **http://127.0.0.1:5173/** を開きます（Python 3 の `http.server`）。

`npx serve` を使う場合:

```bash
npm run preview:cafe:serve
```

Cursor / VS Code では **タスク「Preview: Cafe landing」** または **デバッグ構成「Cafe landing: サーバー起動 + Chrome」**（`.vscode` 参照）でも同様にプレビューできます。
